# Agent Reliability Arena v0.2.0rc1

This is a public **prerelease** of the Agent Reliability Arena release-candidate codebase.

It packages the verified deterministic fixture, provider-free integration layers, private-pilot safeguards, repeated-experiment planning, disclosure-safe export, public showcase verification and career-facing launch materials into one reviewable source state.

## What this prerelease demonstrates

- strict separation between agent reports and independently observed completion evidence;
- same-task comparison between a general policy and a bounded specialist workflow;
- fail-closed role contracts and confined mutation boundaries;
- tamper-evident provider-neutral transport records;
- explicit approval, call, token and monetary-reservation gates before external execution;
- safe private paired-run and repeated-experiment rehearsal without a provider call;
- disclosure-safe public export that retains completed and aborted outcomes;
- byte-pinned public showcase and launch-package manifests;
- source, wheel and clean-install verification on Python 3.10 through 3.13.

## Deterministic fixture results

The included deterministic fixture records:

- General verified outcomes: **2/8**
- Specialist verified outcomes: **6/8**
- False-completion reduction: **3**
- Additional logical calls: **36**

These values validate the software path and evidence plumbing. They are not representative measurements of a hosted or local model.
