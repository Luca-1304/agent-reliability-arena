# CV project entry

## Concise version

**Agent Reliability Arena — Creator / AI Reliability Engineer**  
Designed and shipped a Python evaluation system comparing a general agent with a same-model specialist workflow under identical tasks, tools, failures and evidence rules. Built deterministic request planning, strict role contracts, independent completion verification, tamper-evident ledgers, bounded recovery, safe pause/resume and disclosure-safe public evidence. Verified across Python 3.10–3.13 in source and clean-wheel environments.

## Expanded version

**Agent Reliability Arena — Creator / AI Reliability Engineer**  
Public repository: `github.com/Luca-1304/agent-reliability-arena`

- Framed and implemented a controlled agent-reliability experiment that isolates orchestration as the main changed variable.
- Separated Strategist, Operator, Auditor, Recovery and Synthesiser responsibilities so no single role can plan, mutate, verify and approve completion alone.
- Required exact contract authorisation before file mutation and derived final outcomes from independently observed state rather than agent or tool success claims.
- Built provider-neutral request/result contracts, strict JSON role-output parsing, deterministic call plans and hard pre-call call/token/budget reservation gates.
- Added tamper-evident transport ledgers, preserved abort evidence, verified-prefix continuation and terminal refusal after partial or altered runs.
- Added disclosure-safe export and public replay that retain failed and aborted outcomes while excluding private prompts, raw payloads, local paths and operator notes.
- Built preregistered counterbalanced repeated experiments with named descriptive uncertainty methods and explicit claims limits.
- Published a digest-pinned public showcase and installed verification commands; final matrices pass Python 3.10, 3.11, 3.12 and 3.13, including clean-wheel installation.

## Evidence line

The locked **deterministic fixture** produced 2/8 verified outcomes for the General condition and 6/8 for the Specialist condition, removed three false-completion claims and required 36 additional logical role calls. These figures validate the experiment and evidence plumbing; they are not measurements of an external model.
